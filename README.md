# Py Data Desk

一个可离线打开的 Python 数据处理指令查询网页，覆盖 `pandas`、`numpy`、`seaborn`、`matplotlib` 的常用写法。

## 特点

- 纯静态网页，直接打开 `index.html` 即可使用
- 支持中文和英文关键词搜索
- 支持按库、按任务类型筛选
- 支持按场景流程浏览
- 支持复制代码示例

## 使用方式

1. 下载或克隆仓库
2. 直接打开 `index.html`

不需要安装依赖，也不需要启动本地服务。

## 文件说明

- `index.html`: 页面结构
- `styles.css`: 页面样式
- `script.js`: 页面交互逻辑
- `data.js`: 指令数据与场景配置
